pip

Usage: c4ddev pip [OPTIONS] [ARGS]...

  Invokes Pip in the current Cinema 4D Python distribution. Must be used
  from inside the Cinema 4D applications directory or specified with --c4d.

Options:
  --c4d DIRECTORY
  --help      Show this message and exit.
